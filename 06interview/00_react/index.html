# 什么是 React?

React 是一个开源前端 JavaScript 库，用于构建用户界面，尤其是单页应用程序。它用于处理网页和移动应用程序的视图层。

React 是由 Facebook 的软件工程师 Jordan Walke 创建的。在 2011 年 React 应用首次被部署到 Facebook 的信息流中，之后于 2012 年被应用到 Instagram 上

# react key 作用 为什么不使用 index 做 key 值

react 使用 key 来标识组件，每个 key 对应一个组件，在一定范围内组件的 key 是唯一稳定的，如果 key 值相同 react 会认为他们是同一个组件，后续的组件 react 将不会去创建。

常用场景 数组动态创建组件

涉及到数组的动态变更，例如数组新增元素、删除元素或者重新排序等，使用 index 作为 key 会导致数据错误的展示
